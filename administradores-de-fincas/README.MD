## Administración de fincas. Base de datos en sql server 2022
### Soporte: https://www.linkedin.com/in/oscarlizarragag/

Debes de ejecutar en el SQL Management Studio 22 estos archivos en el mismo orde que estan marcados, en caso contrario puede instalarse mal.

* 0 Base de datos (Crea una base de datos llamada "administracion-fincas", puedes cambiar el nombre si lo deseas, modificando el archivo.
* 1 Tablas de Comunidades y Estructura
* 2 Tablas de Contabilidad y Movimientos
* 3 Tablas Reuniones y Documentacion
* 4 Vista de unidades y propietarios activos
* 5 Vista del Estado de Cuentas por Comunidad y Mes
* 6 Vista de Deudas de Propietarios
* 7 Vista de Movimientos de Gasto Detallado por Proveedor
* 8 Vista de Acuerdos Aprobados Recientes
* 9 Procedimiento Almacenado SP-RegistrarPagoCuota
* 10 Ejecutar el procedimiento para registrar el pago
* 11 Procedimiento Almacenado para Asignación de Gastos Proporcionales
* 12 Función para Obtener la Ruta del Documento más Reciente
* 13 Función para Calcular el Saldo Pendiente del Propietario
* 14 Procedimiento Almacenado para Registrar el Cierre Contable Anual
* 15 Insercion de datosen la base de datos
* 16 Procedimiento para Reporte Contable Detallado
* 17 Gestión de Incidencias y Averías
* 18 Procedimiento para Registrar Incidencia
* 19 Tablas para Gestión de Siniestros (Seguros)
* 20 Tablas para Gestión de Remesas Bancarias (SEPA)
* 21 DetalleRemesa (Relación entre Remesa y Cuotas a Cobrar)
* 22 Trigger para Mantener el Coeficiente de Participación
* 23 Trigger para Actualizar el Estado de Morosidad del Propietario
* 24 Secuencia para Actas de Junta
* 25 Secuencia para Facturas Emitidas (si la comunidad factura algo)
* 26 Índices (Performance)
* 27 Creación de Esquemas (Organización y Seguridad)
* 28 Siniestros y Seguimiento (Esquema Operaciones)
* 29 Secuencias de Numeración
* 30 Índices Clave para Rendimiento
